Kluczowe funkcje Key Features

🔐

Logowanie i rejestracja z weryfikacją CAPTCHA Login and Registration with CAPTCHA verification

Zapewnij bezpieczeństwo dostępu poprzez CAPTCHA podczas logowania i rejestracji. Ensure secure access with CAPTCHA during login and registration.

🎨

Tryb jasny i ciemny Light & Dark Mode

Przełączanie między jasnym i ciemnym motywem dla komfortu użytkownika. Switch between light and dark themes for user comfort.

🌍

Wersje językowe: polska i angielska Language versions: Polish and English

Możliwość wyboru języka polskiego lub angielskiego. Option to choose Polish or English language.

🛠️

Panel administratora Admin Panel

Zaawansowane zarządzanie systemem i użytkownikami z poziomu panelu admina. Advanced system and user management via the admin panel.

👥

Panel tworzenia i zarządzania grupami Group Creation and Management Panel

Twórz i zarządzaj grupami użytkowników w systemie. Create and manage user groups in the system.

✏️

Tworzenie, edycja i organizacja notatek Creating, Editing, and Organizing Notes

Dodawaj, edytuj i porządkuj notatki według własnych potrzeb. Add, edit, and organize notes according to your needs.

📄

Eksport notatek do PDF Export Notes to PDF

Zapisuj notatki w formacie PDF do udostępniania lub archiwizacji. Save notes as PDF for sharing or archiving.

📧

Wysyłanie notatek e-mailem Sending Notes via Email

Udostępniaj notatki bezpośrednio za pomocą e-maila. Share notes directly via email.

📅

Kalendarz z wydarzeniami i terminami Calendar with Events and Deadlines

Zarządzaj wydarzeniami i ważnymi terminami w zintegrowanym kalendarzu. Manage events and important deadlines in an integrated calendar.

🗳️

Głosowanie grupowe na najlepszą notatkę Group Voting for the Best Note

Wspólnie wybieraj najlepszą notatkę poprzez system głosowania. Collectively select the best note through voting system.
